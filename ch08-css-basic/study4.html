-> id는 본명(중복불가), class는 별명(중복가능)이기 때문에 하나의 p태그에 띄어쓰기로 구분만 해주면 클래스가 2개일 수 있음!!(혼합조합 가능)
css에서 p.c{font-size:60px; color:#000;}는 p태그 중에서 클래스 c를 가지고 있는 것만 효과를 부여한다는 의미
css에서 .c는 클래스를 c로 부여받은 모든 애들에게 효과를 주겠다는 의미
css에서 p.a.b는 p태그 중에서 클래스가 a와 b를 모두 가지고 있는 애들에게 효과를 주겠다는 의미
css에서 맨 앞자리에 있는 (. 또는 # 앞에 올 수 있는) 것은 html태그명뿐이다. -> ex) h2.c{} /// p.b{} ///li#a 등등

.box h2{backgruond-color: gray;} -> class명이 box인 태그안에 있는 h2들에게 배경색을 회색으로 주겠다는 의미

div.box h2{backgruond-color: gray;} -> div의 클래스가 box인 태그 안에 있는 h2들에게 배경색을 회색으로 주겠다는 의미

section.box h2{backgruond-color: gray;} -> section의 클래스가 box인 태그 안에 있는 h2들에게 배경색을 회색으로 주겠다는 의미
